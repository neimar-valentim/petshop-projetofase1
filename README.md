# Projeto Petshop Online

Projeto desenvolvido para a disciplina Fundamentos de Sistemas Web, da PUC-RS.

## Objetivo

Desenvolver a primeira etapa de um sistema web para petshop utilizando apenas HTML, conforme os requisitos da atividade.

## Funcionalidades

* Cabeçalho com menu de navegação
* Produtos em três categorias:

  * Acessórios
  * Rações não perecíveis
  * Higiene e limpeza
* Serviços:
  * Banho e tosa
  * Banho e tosa com tele-busca
* Rodapé com informações de contato

## Tecnologia Utilizada
* HTML5
